Функции, которые облегчают работу бизнеса

Мобильное приложение с уникальными возможностями:

  • Подключение к базе данных вашей компании или создание своей базы для более широкого круга предприятий.
  • Интеграция с системами 1С или другими зарубежными платформами через синхронизацию данных в различных форматах (CSV, XML, XLSX, TXT и другие).

Два режима работы:

Кабинет Сотрудника

  • Простая регистрация через ИИН с автоматическим заполнением данных из государственной базы.
  • Отправка заявки на добавление в Кабинет Руководителя.
  • Учет рабочего времени согласно настройкам Руководителя.

Кабинет Руководителя

  • Управление списком сотрудников с возможностью добавления по заявке.
  • Отправка пригласительных ссылок для подключения сотрудников.
  • Гибкая настройка графика рабочего времени, включая перерывы и обеды.
  • Оповещения о приходе/уходе сотрудников.

Учет рабочего времени:

  • Отслеживание местоположения сотрудников относительно координат рабочего места.
  • Уведомления при выходе из зоны работы
  • Активация режима “На работе” через Кабинет Сотрудника.
  • Биометрические данные:
  • Возможность активации по Touch-ID, Face-ID и другим биометрическим функциям смартфона.
  • Обязательная отправка селфи-фото для подтверждения личности.
  • Touch-ID, Face-ID и другие функции смартфона.
  • Использование смартфона как считывателя биометрических данных.

Гарантированная Безопасность для Вашего Бизнеса

Наши услуги по проверке специалистов обеспечивают вас всесторонней информацией о кандидатах, минимизируя риски и создавая безопасное рабочее окружение. Мы предоставляем проверку по следующим параметрам:

  • Гарантия совпадения данных с официальными записями.
  • Подтверждение статуса индивидуального предпринимателя.
  • Проверка наличия штрафов и задолженностей по налогам.
  • Отсутствие в реестре должников.
  • Нахождение в розыске по экономическим преступлениям.
  • Проверка на алименты, связь с финансированием терроризма.
  • Состояние в реестре без вести пропавших.
  • Наличие судебных разбирательств.
  • Акты по административным, уголовным или гражданским делам.
  • Отсутствие в реестре педофилов.
